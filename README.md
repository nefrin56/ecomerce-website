                                                                    eCommerce Website
This is a simple eCommerce website built using Java Server Pages (JSP) and Servlets. It provides basic functionalities like viewing products, adding items to a cart, and checking out.
This project demonstrates how to build a dynamic website with backend integration and a MySQL database.
